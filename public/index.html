CIS*2750 Genealogy API

This genealogy API is written for a computer science course at the University of Guelph. It uses the GEDCOM file format to generate and analyse family tree objects. The API can do the following:

  • Produce a GEDCOM file from a GEDCOM object
  • Produce a GEDCOM object from a GEDCOM file
  • Get a list of ancestors and descendants
  • Add indiviudals to the GEDCOM object
